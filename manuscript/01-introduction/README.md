# Chapter 01: Introduction (도입)

## 챕터 목표
COPYWRITING MADE System의 핵심 가치를 소개하고, 독자의 호기심을 유발하여 다음 챕터로 이끈다.

## 프레임워크
- **선택 프레임워크**: PAS (Problem → Agitate → Solve)
- **주요 프로세스 단계**: Emotion (감정으로 주목시키기)

## 타겟 오디언스
- 마케터, 콘텐츠 크리에이터, 스타트업 대표
- 카피라이팅에 시간과 비용을 투자하지만 성과가 나지 않는 사람들

## 핵심 메시지
> "당신의 콘텐츠 데이터 안에 이미 최고의 카피가 숨어있다. AI가 그것을 찾아드립니다."

## 구성
1. **Problem**: 효과 없는 카피라이팅에 낭비하는 시간과 비용
2. **Agitate**: 카피 하나로 전환율이 300% 달라지는 현실
3. **Solve**: 데이터 기반 AI 카피라이팅의 등장
