# Chapter 02: 3가지 카피라이팅 프레임워크

## 챕터 목표
PAS, BAB, FAB 세 가지 프레임워크를 심층 분석하고, 각 프레임워크의 활용법을 데이터와 사례로 증명한다.

## 프레임워크
- **선택 프레임워크**: FAB (Features → Advantages → Benefits)
- **주요 프로세스 단계**: Logic (논리로 주목 유지하기)

## 타겟 오디언스
- 기본 카피라이팅 개념은 알지만 프레임워크 활용에 어려움을 느끼는 실무자
- 체계적인 카피라이팅 방법론을 찾는 마케팅 팀

## 핵심 메시지
> "올바른 프레임워크 하나가 카피의 전환율을 결정한다. 상황에 맞는 프레임워크를 선택하는 법을 알려드립니다."

## 구성
1. **Features**: PAS, BAB, FAB 각 프레임워크의 구조와 특징
2. **Advantages**: 각 프레임워크가 적합한 상황과 비교 분석
3. **Benefits**: 올바른 프레임워크 선택이 가져오는 실질적 성과
